Repositorio utilizado para albergar tanto los scripts como las muestra de mi TFG. Cada parte se encuentra dentro de su directorio.
El directorio cerradura contiene las muestras en formato complex16s y txt y los scripts realizados para el procesamiento de las muestras.
El directorio tarjeta contiene un script en python utilizado para generar tablas de diffing en latex y en la carpeta muestras todas las muestras de las tarjetas excepto la tarjeta personalizada por motivos de privacidad.
